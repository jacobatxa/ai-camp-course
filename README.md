# 超级 AI 营 · 课程仓库

9–11 岁 AI 素养夏令营课件与教师教案。推荐 **三日营**（每天 90–120 分钟），课件可按天拆分放映。

|------|------|

| 一日营 / 三日营主页 | https://jacobatxa.github.io/ai-camp-course/ai-day-camp/ |

## 课件快捷操作

- **G** — 打开目录，跳转幻灯片  
- **E** — 编辑模式：页内改字，自动保存到本机浏览器（可导出/导入 JSON）  
- **空格 / ← →** — 翻页  

## 目录结构

```
ai-day-camp/
  kids.html      # 孩子版幻灯（29 页）
  design.html    # 三日营教师教案
  parent.html    # 家长版
  assets/bg/     # 本地背景图（不依赖 Unsplash）
```

## 本地预览

```bash
open ai-day-camp/kids.html
# 或
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080/ai-day-camp/kids.html
```
