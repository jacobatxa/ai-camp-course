# 猫挑战视频素材

将以下文件放入本目录后，课件 [kids.html](../kids.html) 的「猜猜哪个是真的」环节即可本地播放。

| 文件名 | 说明 |
|--------|------|
| `cat_daily.mp4` | 01 · 可爱的日常猫 |
| `cat_office.mp4` | 02 · 穿西装上班的猫 |
| `cat_school.mp4` | 03 · 背书包上学的猫 |
| `cat_daily.jpg` | 可选：视频 01 封面 |
| `cat_office.jpg` | 可选：视频 02 封面 |
| `cat_school.jpg` | 可选：视频 03 封面 |

## 建议规格

- 格式：MP4（H.264）
- 时长：每段 10–30 秒
- 分辨率：1280×720 或 1920×1080
- 体积：单文件建议 &lt; 15MB，便于课堂加载

若 MP4 缺失，课件会显示占位提示，仍可完成投票与揭晓环节。
